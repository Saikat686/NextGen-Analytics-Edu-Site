Certified Data Science Professional

The course includes four Azure certifications from Microsoft - advancing through these, one by one, will build your knowledge of core concepts of data science, and how they can be applied to the Microsoft Azure environment.

Course Overview

The Certified Data Science Professional course at Lumify Learn prepares you for a career in the fast-growing field of data science. This 100% online, interactive boot camp spans 3 to 5 months, allowing you to learn at your own pace. Guided by industry experts, you will gain comprehensive knowledge through four Azure certifications from Microsoft. These certifications will build your understanding of core data science concepts and their applications within the Microsoft Azure environment.

What will I learn in this course?

  • Microsoft Azure Fundamentals: Gain foundational knowledge of cloud concepts, core Azure services, security, privacy, compliance, and Azure pricing and support.
  • Microsoft Azure Data Fundamentals: Learn the basics of database concepts in a cloud environment, cloud data services, and foundational knowledge of Microsoft Azure data services.
  • Microsoft Azure AI Fundamentals: Understand fundamental AI concepts and learn about Azure services used to create AI solutions

Career Pathways

Graduates of the Certified Data Science Professional course can pursue various roles, including:

  • Business Analyst: Analyse and improve business processes using data-driven insights.
  • Data Analyst: Interpret complex data to help organizations make informed decisions.
  • Data Scientist:Develop advanced analytical models to solve complex problems and drive strategic initiatives.
